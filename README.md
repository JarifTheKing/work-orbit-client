# WorkOrbit - Freelance Job Marketplace

**Live Site:** [https://workorbit.vercel.app](https://workorbit.vercel.app)

---

## Bullet Points

- **1. User Authentication:** Secure login and registration using Firebase with email and Google sign-in.

- **2. Job Management:** Users can post, explore, update, and delete job listings easily.

- **3. Theme Toggle:** Switch between light and dark themes using DaisyUI and Tailwind CSS.

- **4. Responsive Design:** The website works smoothly on mobile, tablet, and desktop devices.

- **5. Smooth Animations:** Beautiful transitions and animations using Motion.

- **6. Job Filtering:** Sort and filter jobs by category or posted date/time.

---

**Built with:** React, Tailwind CSS, Firebase, and MongoDB.
