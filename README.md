# WorkOrbit - Freelance Job Marketplace

**Live Site:** [https://workorbit.vercel.app](https://workorbit.vercel.app)

---

## Features

- **User Authentication:** Secure login and registration using Firebase with email and Google sign-in.
- **Job Management:** Users can post, view, edit, and delete job listings easily.
- **Theme Toggle:** Switch between light and dark themes using DaisyUI and Tailwind CSS.
- **Responsive Design:** The website works smoothly on mobile, tablet, and desktop devices.
- **Smooth Animations:** Beautiful transitions and animations using GSAP.
- **Job Filtering:** Sort and filter jobs by category or posted date/time.

---

**Built with:** React, Tailwind CSS, Firebase, and MongoDB.
